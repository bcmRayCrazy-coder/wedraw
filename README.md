# wedraw

Draw a picture on command line!

## Install
```bash
$ git clone https://github.com/bcmRayCrazy-coder/wedraw.git
$ cd wedraw
$ yarn
```

## Project structure
-   `src` Main scripts
    -   `lib` Main lib
    -   `menu` Launch menu
-   `test` Test scripts
    -   `ask` Test src/lib/void/ask.ts
    -   `out` The tested image
-   `dist`  Generated .js files

## build
```bash
$ yarn build
```
